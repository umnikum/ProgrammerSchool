#Вводная часть, класс map

Структура map хранит значения в упорядоченном виде аналогично vector.
Главным отличием является отсутствие индекса элемента. Вместо него используется значение-ключ.
Ключом может быть большинство объектов с которыми мы привыкли работать, которые можно сравнивать между собой.

Удобство использования ключа состоит в том что они не обязаны идти по порядку внутри map.
В случае vector будет необходимо оставлять незаполненными несуществующие индексы памяти,
map изначально хранит только внесённые в неё данные, в отсортированном по ключам виде.
