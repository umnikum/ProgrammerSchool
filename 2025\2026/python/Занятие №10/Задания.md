#Задания

#Разминка

- 0.1) Составьте список из слов и по введённом пользователем индексу выводите слово из списка.

- 0.2) Пользователь вводит по одном номеру: если 0, то программа завершается и выводится список, иначе
номер сохраняется в список.

#Задачи

- 1) Найти среднее арифметическое среди чисел из списка

- 2) Задаются два списка одинакового размера из строк. Вывести их в столбцы.

- 3) Пользователь вводит набор слов через пробелы, создайте список содержащий только из введённых чисел
